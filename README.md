# Internship Finder Platform

Проект представляет собой веб-платформу для поиска стажировок и начальных вакансий для студентов и выпускников.  
Пользователи могут просматривать предложения, фильтровать их по направлениям и откликаться на интересующие позиции.  
Компании получают возможность публиковать вакансии и просматривать отклики кандидатов.  
Цель проекта — упростить процесс поиска первой работы и стажировки.  
Разработка ведётся с использованием Agile-подхода и Scrum-процесса.  
В рамках лабораторной работы моделируется один спринт разработки.
