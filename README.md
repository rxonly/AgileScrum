# Internship Finder Platform

Проект представляет собой веб-платформу для поиска стажировок и начальных вакансий для студентов и выпускников.  
Пользователи могут просматривать предложения, фильтровать их по направлениям и откликаться на интересующие позиции.  
Компании получают возможность публиковать вакансии и просматривать отклики кандидатов.  
Цель проекта — упростить процесс поиска первой работы и стажировки.  
Разработка ведётся с использованием Agile-подхода и Scrum-процесса.  
В рамках лабораторной работы моделируется один спринт разработки.

## Sprint Review

В ходе спринта были завершены задачи по регистрации и авторизации пользователей.  
Также начата реализация отображения списка стажировок и пользовательского интерфейса главной страницы.  
По итогам спринта платформа получила базовый функционал для работы пользователей.  
Задача по фильтрации вакансий не была завершена и перенесена в следующий спринт.

## Sprint Retrospective

Хорошо прошло планирование задач и распределение приоритетов.  
Команда смогла реализовать ключевые функции в срок.  
Возникли сложности с оценкой объёма работ по фильтрации данных.  
В следующем спринте стоит разбивать сложные задачи на более мелкие и точнее оценивать их сложность.
